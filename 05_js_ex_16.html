1. jQuery 함수

    - jQuery() = $()

    - 함수의 인수(인자)를 기준으로 DOM 또는 HTML 문자열과 일치하는 요소(Element)의 컬렉션(객체와 비슷함)을 반환함.

  2. jQuery 진입점(실행 시작점)

    - $(document).ready(function(){ 실행구문 });

    - $(function(){});

    - 진입점(시작점) : DOM이 모두 로딩된 이후에 코드를 실행할 수 있게 함( 이미지 로딩은 제외 )

    - DOM이 먼저 로딩이 완료되고, 이후에 이미지 로딩이 완료됨.
